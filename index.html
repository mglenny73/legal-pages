Glenny Home Assistant Integration

This is a private, personal-use application that connects a self-hosted Home Assistant instance to Google's Smart Device Management API for a single household's Nest devices (cameras and thermostats). It is not a public product, is not distributed to other users, and is not monetized.

Privacy Policy

This application reads device status (camera events, thermostat state) from the Google Smart Device Management API solely to display and automate that data within a private, self-hosted Home Assistant instance on the household's own local network.

  • Data retrieved from Google's API is used only for local home automation and is not shared with, sold to, or transmitted to any third party.
  • No data is used for advertising, analytics, or any purpose beyond operating the household's own smart home devices.
  • Authentication tokens are stored only within the household's own private Home Assistant instance and Google Cloud project.
  • Access can be revoked at any time via the household's Google Account permissions page.

Terms of Service

This application is provided for personal, non-commercial, single-household use only. It is used solely to integrate the owner's own Nest devices with their own Home Assistant instance. There are no other users, no public sign-up, and no service availability guarantees beyond the owner's own infrastructure.
